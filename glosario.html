Glosario

Internet:
grupo grande de computadoras conectadas una con otras a través de cables, líneas telefónicas y otros tipos de cableado tecnológico.

Protocolo:
sistema de reglas que gobierna el comportamiento de algún sistema.

Router:
dispositivo de transmisión de datos entre dos redes diferentes.

Servidores:
máquinas que almacenan o albergan todos los recursos.

Cliente:
dispositivos que utilizamos de forma personal como las laptops, los smartphones, etc.

Navegador:
programa que nos permite visualizar las páginas web en nuestra computadora.

Dirección ip:
código único que identifica un equipo conectado a una red.

Hypertext Markup Language:
conjunto de instrucciones estandarizadas que dictan cómo el contenido de una página web debe ser organizado y presentado por una aplicación cliente (i.e., web browser).

Head:
contiene información descriptiva sobre el documento en sí, como su título, las hojas de estilo (CSS) que utiliza, los scripts y otros tipos de información "meta".

Body:
contenido visible que aparece en la página web.
